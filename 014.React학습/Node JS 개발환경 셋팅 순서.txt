[ Node JS 개발환경 셋팅 순서 ]

- 뷰JS, 리액트 공통

1. 깃허브 소스 클론하기
2. 노드 cli 환경 설치하기
(1) 기존 프로젝트 폴더 이름 바꾸기
ex. aaa -> aaa2 (탐색기에서 직접 변경, vscode에서 변경 x)
(2) vscode 열기
(3) 깃허브에 올린 이름과 같은 프로젝트 생성
(4) 프로젝트에 들어가서 확장 의존성 프로그램 설치
(5) vscode 닫고 탐색기에서 이름 바꾼 폴더의 개발 소스만 지우고 복사 붙여넣기
(6) 이름 바꾼 백업 파일 지우기
(7) 다시 cli 환경에서 npm run dev로 로컬 서버 실행 시켜서 동일하게 작동됨을 확인
