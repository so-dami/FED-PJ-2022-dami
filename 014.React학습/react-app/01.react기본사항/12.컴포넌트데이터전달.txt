[ 리액트 컴포넌트에서 데이터 전달하기 ]

일반적으로 리액트에서 데이터는 하위 컴포넌트로 props를 통하여 전달함~~
-> 일반적인 함수는 바깥 쪽에 선언하면 전역변수, 함수 내부는 지역 변수로 사용했으나 컴포넌트는 자식 컴포넌트가 부모 컴포넌트의 변수, 함수 등을 직접 호출할 수 없음

1. 부모 컴포넌트 -> 자식 컴포넌트로 값 전달
->>> Props Down
->>> 속성을 통하여 값을 전달함

<컴포넌트명 속성명=값 />

2. 자식 컴포넌트 -> 부모 컴포넌트로 값 전달(함수 호출)
->>> Props Function Down & Up -> 함수를 내려보내고 호출 받음

<컴포넌트명 속성명={부모함수명}>
-> 자식 컴포넌트에서 속성으로 보낸 부모 함수를 호출할 수 있음

3. 형제 컴포넌트는 서로 값을 어떻게 전달하나?
자식 -> 부모 -> 다른 자식
