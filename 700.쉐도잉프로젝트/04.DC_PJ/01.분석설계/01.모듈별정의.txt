2. 메뉴 버튼 모듈
    
    1) 모듈명: MenuBtn(JS, CSS 이름도 동일하게)

    2) 구조 정의
        Root > section > img Box + title Box + button Box

    3) 객체화
        {
            "isrc": "이미지 경로",
            "tit": "(작은 타이틀)^큰 타이틀",
            "btn": "버튼 문구",
            "link": "이동 카테고리",
        }

        -> "tit" 속성은 ^로 데이터를 구분함
        -> 컴포넌트별 객체화 데이터 반영은 기본임

    4) 기능 정의

        (1) 버튼 오버 시 하이라이트
            - 이벤트: mouseover, mouseout
            - CSS
        
        (2) 버튼 클릭 시 페이지 이동
            - 이벤트: click
            - location: route 이동
