2. 메뉴 버튼 모듈
    
    1) 모듈명: MenuBtn(JS, CSS 이름도 동일하게)

    2) 구조 정의
        Root > section > img Box + title Box + button Box

    3) 객체화
        {
            "isrc": "이미지 경로",
            "tit": "(작은 타이틀)^큰 타이틀",
            "btn": "버튼 문구",
            "link": "이동 카테고리",
        }

        -> "tit" 속성은 ^로 데이터를 구분함
        -> 컴포넌트별 객체화 데이터 반영은 기본임

    4) 기능 정의

        (1) 버튼 오버 시 하이라이트
            - 이벤트: mouseover, mouseout
            - CSS
        
        (2) 버튼 클릭 시 페이지 이동
            - 이벤트: click
            - location: route 이동

3. 비디오 소개 모듈

    1) 모듈명: VidIntro(JS, CSS 이름도 동일하게)

    2) 구조 정의
        Root > video Box > vidoe + img + title Box > heading + p

    3) 객체화
        {
            "vsrc": "유튜브 동영상 경로",
            "isrc": "포스터 이미지 경로",
            "stit": "작은 타이틀",
            "btit": "큰 타이틀",
            "sum": "요약 소개",
            "desc": "세부 소개",
            "link": "관련 링크 정보",
        }

    4) 기능 정의

        (1) 동영상 영역 클릭 시 재생, 멈춤  

        (2) 관련 링크 새창 열기

4. 비디오 스와이프 모듈

    1) 모듈명: VidSwipe (JS, CSS, data)

    2) 구조 정의
        스와이퍼 모듈 + 비디오 재생 모듈

    3) 객체화
        {
            isrc: "",
            vsrc: "",
            cat: "",
            tit: "",
        }

        (1) 이미지 경로: isrc
        (2) 유튜브 URL: vsrc
        (3) 카테고리: cat
        (4) 제목: tit

    4) 기능 정의
    
        (1) 비디오 메뉴 스와이프 기능(스와이퍼)

        (2) 비디오 메뉴 클릭 시 별창 동영상 재생(+ 닫기 버튼 구현)

5. 캐릭터 모듈

    1) 모듈명: Cat (JS, CSS, data)

    2) 구조 정의
        제목 + 스와이퍼 모듈
        -> 하위 스와이퍼 모듈(이미지 + 이름)

    3) 객체화
        {
            idx: "",
            tmsrc: "",
            cname: "",
            cdesc: "",
            facts: "",
            alignment: "",
            type: ""
        }

        (0) 인덱스 번호: idx (유일키 == 기본키)
        (1) 썸네일 이미지 경로: tmsrc
        (2) 캐릭터 이름: cname
        (3) 캐릭터 설명: cdesc
        (4) 캐릭터 명세: facts
        (5) 캐릭터 분류: alignment(히어로/빌런/복잡)
        (6) 캐릭터 타입: type(개인/단체)

    4) 기능 정의
    
        (1) 캐릭터 메뉴 스와이프 기능(스와이퍼)

        (2) 캐릭터 메뉴 클릭 시 캐릭터 상세페이지로 이동(라우터)

6. 캐릭터 상세페이지 모듈: 페이지 구성 모듈

    1) 모듈명: Detail (JS, CSS, data)

    3) 객체화
