[ 뷰JS 적용하기 ]

1. 공통 적용을 위한 페이지 구조화하기
    
    1) 데이터가 달라지는 부분 외엔 모두 같아지게 함
    -> 남자, 여자, 스타일 3개의 페이지 변경 시

        (1) 이미지: 폴더만 다르고 이름은 같음
        -> 뷰JS에서 v-bind:src="'같은경로이름'+변수"
        -> 변수는 Vuex.Store 사용
        -> 변수 업데이트는 mutations 메서드로 작성
        -> 확장자 일치: 여기서는 png로 일치함

        (2) 데이터: 분류별 코드를 객체화하여 데이터를 구조화함
        -> 3가지 분류별 객체를 만들고 데이터키를 속성명으로 생성하여
        구분되도록 만들고 사용하기 쉽게 함
        -> 뷰JS에서 각 요소의 특정 위치에 표시되는 데이터값을 매칭하여
        v-html / v-text / {{}}로 화면에 출력
        -> 따라서 데이터 변경부분 ㅗ이엔 요소의 구조가 모두 같게 함

    2) 반복되는 요소와 데이터는 변경사항에 맞게 객체데이터를 활용하여
    반복구조를 생성함

        (1) 뷰JS에서는 v-for 속성구문과 v-if 조건구문을 적절히 사용하여
        요구사항에 맞는 화면이 구성되게 함
        
        (2) 이때 v-for를 쓴다면 반드시 v-bind:key를 사용할 것
            -> 키로 활용할 데이터는 v-for="(v,i) in 객체/배열"
            -> i는 배열일 경우 순번, 객체일 경우 속성명으로 유일키로 간주
